# GSIcon WebComponent
 
GSIcon WebComponent is a wrapper around standard HTMLForm element with addtional event handling.
  
<br>
 
## Attributes
---
 
| Name               | Description                                              |
|--------------------|----------------------------------------------------------|
| name               | Bootstrap icons name                                     |
| size               | Icon size (1..5)                                         |

<br>
 
## Example
---

```HTML
<gs-icon name="alarm" size="2"></gs-icon>
```
