# GSCSSMap Class
 
GSCSSMap Class is a simple Wrapper around computesdStyledMap to support both, the Firefox and WebKit Mode.
 
<br>

## Usage 

```JavaScript
const css = GSCSSMap.getComputedStyledMap(document.body);
const val = css.get('width');
```
